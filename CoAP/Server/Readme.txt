## General info

	Server for CoAP
 This is a server capable of communicating with a client and communication of data is in JSON format.


## Technology used

* Python 2.7 

* CoAPthon library used for CoAP protocols (https://github.com/Tanganelli/CoAPthon)

* json library


## Setup

To run this project, install the CoAPthon to a new python 2.7 environment, change the sys.path in the source code to the new environment folder. using command prompt run the code, this will activate the server wich will listen to the specified port.
